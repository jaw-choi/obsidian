# 📌 프로젝트: 콘솔 뱀서라이크

## 1. 프로젝트 개요
- **소개:** 콘솔 환경에서 뱀서류(서바이버) 핵심 루프를 구현한 프로토타입.
- **기간:** 2026-02-04 ~ 2026-02-10
- **기술 스택:** C++, Windows Console, Object Pooling, Quadtree, AABB.

## 2. 주요 기능
- **Player:** 이동, 자동 공격 시스템, 스탯 시스템.
- **Enemy:** 스폰 및 추격 로직, 경험치 드랍.
- **최적화:** 오브젝트 풀링(Object Pooling), 공간 분할(Quadtree).

## 3. 트러블슈팅
- **메모리 증가 문제:** `prewarm()`이 매 틱마다 호출되어 프로세스 메모리가 무한 증가하던 버그 해결.
- **충돌 누락:** Quadtree 전환 후 죽은 객체까지 검사하던 문제 해결.

## 4. 회고
- 엔진 루프 관점에서 문제를 추적하며 프로젝트 수행 역량을 확보함.
- 자료구조 기반 최적화(Quadtree)의 중요성을 체감함.
